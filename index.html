    This is an alternative program guide for MidAmeriCon II. It should be suitable for use on most browsers and devices. It's an instance of KonOpas, an open-source project providing conventions with easy-to-use mobile-friendly guides. In addition to KonOpas, the schedule is also available as a mobile app, Grenadine, and a web page.

    Program and participant data were last updated .

    This guide will work in your browser even when you don't have an internet connection.

    You can also install it as a home screen app

    Safari on iPhone/iPad
    Tap on (share), then tap Add to Home Screen.
    Chrome on Android
    Tap on (menu), then tap Add to Home Screen.
    Firefox on Android
    Tap on (menu), then tap Bookmark and select Options, and Add to Home Screen.
    IE on Windows Phone
    Tap on (menu), then tap Pin to Start
